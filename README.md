`owner`: nomor WhatsApp anda.  
`prefix`: prefix bot.  
`uaOverride`: your user agent.  
`Barbar`: Mhankbarbar API. (https://mhankbarbar.moe/)  
`lol`: lolhuman API. (https://lolhuman.herokuapp.com/)  
`Vhtear`: Vhtear API. (https://api.vhtear.com/)
